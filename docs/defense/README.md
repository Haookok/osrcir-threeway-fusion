# 答辩材料目录

本文件夹存放**本科毕业设计答辩**所需的全部材料，含两套风格的 PPT（可自由选择）和完整的 Q&A 文档。

## 文件清单

| 文件 | 说明 | 推荐场景 |
|:--|:--|:--|
| `defense_slides.pdf` | **LaTeX Beamer 版答辩 PPT**（19 页，PDF） | 正式答辩推荐 |
| `defense_slides.tex` | Beamer 源码，可修改后重新编译 | 需要微调时 |
| `defense_QA.md` | **答辩 Q&A 文档**（23 个问题 + 数据速查表） | 答辩前复习 |
| `defense.pptx` | python-pptx 版本（16 页，可在 PowerPoint 中编辑） | 需要现场手改时 |
| `generate_defense_ppt.py` | pptx 生成脚本 | 重新生成 .pptx |
| `答辩问题与回答指南.md` | 早期版本 Q&A（40+ 问题） | 补充参考 |
| `_legacy/` | 更早的答辩草稿归档 | — |

## 推荐答辩工作流

**主打组合**：`defense_slides.pdf` + `defense_QA.md`

1. **开场**：用 `defense_slides.pdf` 答辩陈述（15 分钟，19 页）
   - 字体、样式与毕业论文同源（都是同济蓝），视觉连贯
   - 复用了论文中的 7 张图，无需重新设计
   - 附录已预留 2 页常见问题 slides，用于现场提问时调取
2. **复习**：熟读 `defense_QA.md`
   - 按 5 大类组织：选题/方法/实验/结果/工程
   - 附"关键数据速查表"和"15 分钟时间分配建议"
3. **备选**：如需 PowerPoint 格式（方便现场改动画/加视频），用 `defense.pptx`

## defense_QA.md 问题清单

按 5 大类共 23 个问题：

1. **选题与创新性**（Q1–Q4）—— 必答区
2. **方法细节**（Q5–Q9）—— 核心区
3. **实验设计**（Q10–Q14）—— 硬核区
4. **结果与分析**（Q15–Q19）—— 结果区
5. **工程与可复现性**（Q20–Q23）—— 工作量区

每个问题都提供详细回答要点与表格/数据支持。末尾附：
- **关键数据速查表**（主实验、消融、Prompt 对比）
- **15 分钟时间分配建议**
- **开场白 / 收尾语模板**

## 重新编译 Beamer PDF

```bash
cd docs/defense/
xelatex defense_slides.tex
# 输出: defense_slides.pdf
```

依赖：TeXLive（`xelatex` + `beamer` + `ctex`），已在 Linux 服务器验证可用。

## 重新生成 pptx（可选）

```bash
pip install python-pptx
cd docs/defense/
python3 generate_defense_ppt.py
```
