# 答辩材料包

本目录包含毕业答辩所需的全部材料：

| 文件 | 说明 |
|------|------|
| `毕业答辩_OSrCIR三路融合.pptx` | 答辩 PPT（17 页，含封底） |
| `generate_defense_ppt.py` | PPT 生成脚本（基于 python-pptx） |
| `答辩QA预案.md` | 32 个预案问题及标准话术（高/中/低概率分档） |
| `实验细节.md` | 实验流程、参数、失败记录、复现指南 |

## 快速复用

重新生成 PPT：
```bash
pip install python-pptx pillow
python generate_defense_ppt.py
```

PPT 模板样式：同济紫 + 深蓝 + 白，16:9 分辨率。修改配色请改 `generate_defense_ppt.py` 顶部的 `TJ_PURPLE / DEEP_BLUE / ACCENT` 等常量。

## 答辩建议流程（总 15 分钟）

| 时间 | 内容 | PPT 页码 |
|------|------|----------|
| 0:00 - 0:30 | 自我介绍 + 主题 | P1 |
| 0:30 - 1:30 | 目录 + 背景 | P2-P3 |
| 1:30 - 3:00 | 任务痛点与改进切入 | P4 |
| 3:00 - 7:00 | **方法核心**（三路融合 + 三个创新） | P5-P8 |
| 7:00 - 10:00 | **实验结果**（主结果 + FIQ 细化 + 相对提升） | P9-P11, P14 |
| 10:00 - 12:30 | 消融实验（网格搜索 + 组件消融） | P12-P13 |
| 12:30 - 14:30 | 贡献总结 + 局限与未来 | P15-P16 |
| 14:30 - 15:00 | 感谢 + 致谢 | P17 |

## Q&A 环节应对

- **先在心里默念 Q&A 预案中的"开场必备口径"**，保证一句话就能把主题和结果讲清；
- **预案分三档**：高频问题（Q1-Q24）务必熟读；中等概率（Q25-Q27）知道关键点；冷门（Q28-Q32）有数据支撑即可；
- **核心数字一页纸**（QA 预案最后）建议背下来。
