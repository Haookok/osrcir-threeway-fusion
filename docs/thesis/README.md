# 论文草稿与版本说明

本目录用于存放 OSrCIR 毕设论文的草稿、模板和版本文件。

## 文件说明

- `thesis.md`：原有论文草稿，保持不动，作为当前原始稿保留。
- `generic_undergraduate_template.md`：通用本科毕业论文模板骨架，便于后续迁移到学校 Word 模板。
- `versions/`：版本化保存的论文稿，每次较大修改建议新建一个版本文件，不直接覆盖旧版本。

## 当前版本策略

为满足“保留原始文件和论文版本”的要求，采用以下约定：

1. 不直接覆盖 `thesis.md`。
2. 新写的稿件统一放在 `versions/` 下。
3. 版本文件名使用 `v日期-说明.md` 的形式。

## 当前已整理版本

- `versions/v2026-04-14-generic-undergraduate-draft.md`

## 后续建议

- 拿到同济大学官方 Word 模板后，以 `generic_undergraduate_template.md` 的章节结构为映射参考。
- 保留 `thesis.md` 作为内容源稿，把学校格式化后的版本继续放到 `versions/` 中，例如：
  - `v2026-04-15-tongji-word-outline.md`
  - `v2026-04-16-tongji-full-draft.md`
