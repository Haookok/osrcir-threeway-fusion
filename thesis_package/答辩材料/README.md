# 答辩材料目录

本文件夹存放**本科毕业设计答辩**所需的全部材料。

## 文件清单

| 文件 | 说明 | 用途 |
|:--|:--|:--|
| `defense.pptx` | 答辩 PPT（16 页，16:9 宽屏） | 答辩主陈述 |
| `generate_defense_ppt.py` | PPT 生成脚本（python-pptx） | 修改 / 重新生成 PPT |
| `答辩问题与回答指南.md` | 40+ 常见问题 + 标准回答 | 答辩前复习 |
| `_legacy/` | 早期版本 PPT 与文档归档 | 参考历史迭代 |

## PPT 结构（16 页）

| 页 | 内容 | 时长建议 |
|:-:|:--|:-:|
| 1 | 封面 | 15 s |
| 2 | 目录 | 15 s |
| 3 | 研究背景与意义 | 60 s |
| 4 | 相关工作与研究问题 | 60 s |
| 5 | 本文主要工作（三层创新） | 45 s |
| 6 | 方法总体框架图 | 90 s |
| 7 | 创新点 1：视觉代理机制 | 60 s |
| 8 | 创新点 2：反幻觉精炼 (V7) | 75 s |
| 9 | 创新点 3：描述融合 | 60 s |
| 10 | 实验设置 | 30 s |
| 11 | 主要结果（9 数据集对比） | 75 s |
| 12 | 消融：Prompt 演进 | 45 s |
| 13 | 消融：α/β 参数 | 30 s |
| 14 | 贡献总结 | 30 s |
| 15 | 局限与未来工作 | 30 s |
| 16 | 致谢 | 15 s |

**总时长约 11 分钟**，符合本科答辩 10-12 分钟常规要求。

## Q&A 文档结构

`答辩问题与回答指南.md` 按类别组织：

1. **选题与动机类**（Q1–Q3）—— 破题必问
2. **方法与创新点类**（Q4–Q10）—— 核心区
3. **实验与数据类**（Q11–Q18）—— 硬核区
4. **工程与技术细节类**（Q19–Q25）—— 证明工作量
5. **局限与展望类**（Q26–Q30）—— 谦虚但不虚

另附：**开场白模板** / **总结语模板** / **尴尬时刻退路** / **答辩前 3 天清单**。

## 重新生成 PPT

```bash
pip install python-pptx
cd docs/defense/
python3 generate_defense_ppt.py
# 输出: defense.pptx
```
