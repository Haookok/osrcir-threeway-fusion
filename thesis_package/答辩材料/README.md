## 使用建议

1. **答辩前 3 天**：通读 `答辩问答准备.md`，把每个问题的 30 秒核心回答说三遍
2. **答辩前 1 天**：
   - 完整过一次 PPT（自己讲 20 分钟 + 录音回放检查）
   - 熟悉 `实验细节汇总.md` 中的关键数值（β=0.7, α=0.9, 35/35, 140 元）
3. **答辩当天**：
   - 提前 30 分钟到场试 PPT（排查字体、动画问题）
   - 带**备份**：PPT 存 U 盘 + 邮箱备份
   - 笔记本随身携带 `答辩问答准备.md` 纸质打印

## 重新生成 PPT

如需修改内容后重新生成：

```bash
cd thesis_package/答辩材料
python3 generate_defense_ppt.py
```

依赖：`pip install python-pptx`

图片从 `../图表及代码/` 读取，改图后直接重跑即可。

## 答辩心态 Tips

- **沉着**：35/35 全部正向是硬数据，敢于表达
- **诚实**：不会的直接说"这点我没深入研究，请老师指教"
- **倾听**：听完老师完整问题再答，不要抢答
- **支撑**：多报具体数值（CIRCO +5.05、β=0.7）比主观判断有说服力
- **局限**：主动承认局限比被戳穿要好，并给出可能的改进方向
